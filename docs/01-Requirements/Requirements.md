## Module Requirements
The following sections document the requirements that the this module need to fulfills to successfully work with the rest of the system as a whole.

| **Requirement Description** | **Measure of<br> Threshold** | **Target<br>Measure** |**Stretch<br>Requirement<br>(Y-N)**|
|-----------------------------| ----------------- | ----------------- | :-----: |
| Surface mounted, 3.3V switching power regulatore | 3.2 Volts | 3.3 Volts | No |
| Surface mounted microcontroller | 1 PIC or ESP | Esp-32 | No |
| Wireless Communication | Able to send or receive a Wi-Fi data | Send and receive Wi-Fi Data to MQTT | Yes |
| Wired communication | Able to send or receive wired data | Send and receive wired data through 8-pin header | No |
| Serial pressire sensor | Able to read the ambient pressure in the area around it | Reads the ambient pressure of the area around it | No |
